# CÓMO CONTRIBUIR

Para contribuir en este proyecto hace falta tener una cuenta de GitHub, vea [Contribuir a proyectos](https://docs.github.com/es/get-started/quickstart/contributing-to-projects?tool=desktop) en la documentación de GitHub para una guía de como extender el proyecto.

En caso de no ser familiar con la plataforma GitHub u otros sistemas Git, [esta guía](https://docs.github.com/es/desktop/overview/getting-started-with-github-desktop) enseña a dar los primeros pasos con la aplicación [GitHub Desktop](https://desktop.github.com/), disponible para sistemas operativos Windows y MacOS.

El proyecto se basa en una serie de archivos Markdown, aprovechándose del renderizado integrado de la página. Igualmente GitHub ofrece un tutorial sobre [Sintáxis de escritura y formato básicos](https://docs.github.com/es/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) de su dialecto de Markdown.

Se busca almacenar únicamente enlaces a recursos externos, más que alojar los propios recursos. En caso de disponer del medio únicamente de formar local, es preferible alojarlo en otro servidor a intentar adjuntarlo directamente. De todas formas, el repositorio debería impedir añadir cualquier archivo que no sera del formato Markdown (.md).
